# Simple Twitter
## Requests Supported
- Follow request
  ```<user 1> follow <user 2>```
- Unfollow request
  ```<user 1> unfollow <user 2>```
- Like request
  ```<user 1> like <user 2> <post_id>```
- Unlike request
  ```<user 1> unlike <user 2> <post_id>```
- Comment request
  ```
  <user 1> comment <user 2> <post_id>
  <text>
  ```
- Uncomment request
  ```<user 1> uncomment <user 2> <post_id> <comment_id>```
- Post request
  ```
  <user> post 
  <title>
  <tag 1> (tags begin and end with #)
  ...
  <tag n>
  <text>
  ```
- Delete request
```<user 1> delete <post_id>```
- Refresh request
```<user 1> refresh```
- Visit request
```<user 1> visit <user 2>```
- Trending request
```trending <n>```
